Discover open source through documentation

Become an open source contributor, help open source projects, and set the standard for positive, inclusive collaboration.

Canonical’s Open Documentation Academy (CODA) is a collaboration between Canonical’s documentation team and documentation newcomers, experts, and everyone in between.

  • Ask to be assigned the task

    About the Academy

    If you’re a newcomer, we can provide help, advice, mentorship, and a hundred different tasks to get started on. If you’re an expert, we want to create a place to share knowledge, a place to get involved with new developments, and somewhere you can ask for help on your own projects.

    A key aim of this initiative is to help lower the barrier into successful open-source software contribution, by making documentation into the gateway.

    About our community

    Our community is you, and we want everyone to feel welcome. Whether this is your first open source adventure, or you're a gnarly cynical veteran, everyone is invited to contribute and to get involved. Newcomers can start on any task they feel interested in, while more experienced members of our community can help guide them.

    We also encourage other open source projects to get involved, and it's our aim to establish the standard for positive, open, inclusive, and friendly collaboration, regardless of the language, and for the betterment of all open source.
